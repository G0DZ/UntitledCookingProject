На текущем этапе (итерация 2) после развертывания приложения для проверки его работоспособности необходимо следующее:
1. Убедиться, что информация о рецепте по андресу вида http://www.povarenok.ru/recipes/show/000000/ получена корректно. Для этого нужно ввести корректный url в соответствующую строку ввода и оценить результат.
2. Убедиться, что информация с первой страницы поиска по имени (адрес вида http://www.povarenok.ru/recipes/search/?name=%EF%E0%E2%EF%E2%E0%EF) получена корректно. Для этого нужно ввести имя (или часть имени) в соответствующую строку ввода и оценить результат.
2. Убедиться, что информация с первой страницы поиска по ингрединтам (адрес вида http://www.povarenok.ru/recipes/search/?ing=%EF%E0%E2%EF%E2%E0%EF) получена корректно. Для этого нужно ввести список ингредиентов через зяпятую в соответствующую строку ввода и оценить результат.
